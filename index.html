Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media. It is one the core languages of the web and is standardized across Web browsers according to W3C specifications.

In order for one to add styling to an element of a web page; it must be targeted by a CSS Selector. A CSS selector defines which element(s) to target and the set of CSS rules to apply. This documentation gives a general overview of the different types of CSS selectors.

Universal Selector

The CSS universal selector ( * ) selects all elements on a webpage.

The * selector can also select all elements inside another element.

Syntax

              
* { 
  css declarations; 
} 

selector * {
    css declarations;
}

Basic Selectors

Basic Selectors are usually the first set of selectors a student usually learns. These are:

  • Element Selector
  • Class Selector
  • ID Selector

Element Selector

The CSS Element Selector matches elements by node name. In other words, it selects all elements of the given type within a document.

Syntax

              
h1 { 
  css declarations; 
}

Class Selector

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period ( . ) character, followed by the class name.

Syntax

              
.classname { 
  css declarations; 
}

ID Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash ( # ) character, followed by the id of the element.

Syntax

            
#id {
  css declarations;
}

Combinators

A CSS selector can contain more than one simple selector. Between these selectors, one can include a combinator.

There are four different combinators in CSS:

  1. Descendant Selector
  2. Direct Child Selector
  3. Adjacent Sibling Selector
  4. General Sibling Selector

Descendant Selector

The descendant combinator, typically represented by a single space character, combines two selectors such that elements matched by the second selector are selected if they have an ancestor element matching the first selector.

Syntax

            
            
div p  {
  css declarations;
}

Direct Child Selector

The direct child combinator ( > ) is placed between two CSS selectors. It matches the elements of the second selector that are the direct children of elements matched by the first. It does not match descendent elements further down the document hierarchy.

Syntax

            
div > p {
  css declaration
}

Adjacent Sibling Selector

The adjacent sibling selector ( + ) is placed between two CSS selectors. It returns only those elements matched by the second selector that are the immediate sibling element of the first selector.

Syntax

            
div + p {
  css declarations;
}

General Sibling Selector

The general sibling selector selects all elements that are siblings of a specified element. If one wants to select siblings of an element even if they are not directly adjacent, then the general sibling combinator can be used ( ~ ).

Syntax

            
div ~ p {
  css declarations;
}

Pseudo Selectors

Pseudo Classes

A CSS Pseudo Class is a keyword added to a selector that specifies a special state of the selected element(s). For example, :hover can be used to change a button's color when the user's pointer hovers over it.

There are numerous Pseudo Classes, a list of the most common will be provided in this documentation. Click the following link to the MDN Pseudo Class Documentation to see a complete list.

Syntax

            
selector:pseudo-class {
  css declarations; 
}
Commonly Used Pseudo Classes
Selector Example Example Description
:active a:active Selects the active link
:checked input:checked Selects every checked input element
:focus input:focus Selects the input element that has focus
:hover a:hover Selects links on mouse over
:first-child p:first-child Selects every p elements that is the first child of its parent
:first-of-type p:first-of-type Selects every p element that is the first p element of its parent
:link a:link Selects all unvisited links
:nth-child(n) p:nth-child(2); Selects every p element that is the second child of its parent
:nth-of-type(n) p:nth-of-type(2) Selects every p element that is the second p element of its parent

Pseudo Elements

A CSS pseudo-element is used to style specified parts of an element. There are numerous Pseudo Elements, a list of the most common will be provided in this documentation. Click the following link to the MDN Pseudo Elements Documentation to see a complete list. Pseudo Elements can be used to:

  • Style the first letter, or line, of an element.
  • Insert content before, or after, the content of an element. 
            
selector::pseudo-element {
  css declarations;
}
Commonly Used Pseudo Elements
Selector Example Example Description
::after p::after Insert something after the content of each p element
::before p::before Insert something after the content of each p element
::first-letter p::first-letter/td> Selects the first letter of each p element
::first-line p::first-line Selects the first line of each p element
